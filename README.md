# UNDO - Reset Your Mind. Rebuild Your Life.

A 100% free, web-based recovery app designed to help students and individuals without financial means break free from porn addiction through science-backed recovery tools, community support, AI-powered coaching, and a distraction-free, calming UI/UX.

## Features

- **Emergency Urge Support**: Panic button with instant calming tools and interventions
- **Streak Tracking**: Visualize progress and celebrate recovery milestones
- **AI Recovery Coach**: Personalized 24/7 AI mentor (Melius™) for tailored guidance
- **Anonymous Community**: Support groups and peer accountability circles
- **Habit Building**: Dopamine reset techniques and recovery skill development
- **Offline Mode**: Core features accessible without internet connection

## Technology Stack

- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
- **AI**: OpenAI GPT-4 API for personalized coaching
- **Deployment**: Render.com (Free Tier), MongoDB Atlas
- **Design**: iOS 26 Liquid Glass design system

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- MongoDB Atlas account (free tier)

### Installation

1. Clone the repository
2. Copy environment files and configure variables
3. Install dependencies: `npm run install:all`
4. Start development: `npm run dev`

### Project Structure

```
UndoV3/
├── frontend/          # Next.js frontend application
├── backend/           # Express.js REST API

├── docs/             # Documentation

```
